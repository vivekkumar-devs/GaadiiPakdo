## 📌 Overview
GaadiiPakdo is a **real-time transport tracking Android application** that enables users to locate nearby vehicles while allowing drivers to broadcast their live location.

---

## 🔧 Key Features

- 🔄 **Single app with role-based access (User & Driver)**
- 📍 **Real-time GPS tracking using Firebase**
- 🗺 **Live map integration (OSMDroid)**
- 📏 **Distance & ETA calculation**
- 🚘 **Driver profile system**
- ⚡ **Continuous real-time data synchronization**
