## 🚗 GaadiiPakdo — Real-Time Transport Tracking System

**GaadiiPakdo** is a role-based Android application that enables real-time visibility of local transport by connecting drivers and users through continuous GPS synchronization.

---

## 📌 Problem Statement
Local public transport systems lack real-time tracking, leading to uncertainty in availability and increased waiting time for commuters.

---

## 💡 Solution
A single Android application with **role-based access**:
- **Drivers** broadcast live GPS location  
- **Users** discover and track nearby vehicles in real time  

---

## ⚙️ Core Capabilities

- **Real-time location streaming** using Firebase Realtime Database  
- **Role-based architecture** (User / Driver within one app)  
- **Live map rendering** using OSMDroid  
- **Distance and ETA estimation**  
- **Continuous background location updates**  
- **Lightweight and responsive UI**
