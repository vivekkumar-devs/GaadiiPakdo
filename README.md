## 📌 Overview

GaadiiPakdo is a **real-time, role-based Android system** designed to improve local transport visibility by connecting **drivers and users through continuous GPS synchronization**.

---

## ⚡ Core Features

- 🔄 **Single app with role-based access (User & Driver)**
- 📍 **Live GPS tracking using Firebase Realtime Database**
- 🗺 **Dynamic map rendering (OSMDroid)**
- 📏 **Distance & ETA calculation**
- 🚘 **Driver profile management**
- ⚙️ **Continuous real-time data updates**
